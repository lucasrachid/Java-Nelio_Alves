Tipos primitivos JAVA
Tipos Numéricos - byte (-128 até 127), short (-32768 até 32767), int (-2147483648 até 2147483647), long (-9223372036854770000 até 9223372036854770000)
Tipos numéricos com ponto flutuante - float (-1.4024E-37 até 3.4028E+38), double (-4.94E-307 até 1.79E+308)
Um caractere Unicode - char ('\u0000' até '\uFFFF')
Valor verdade - boolean {false, true} - Padrão false
String - Cadeia de caracteres (Palavras ou textos)

%n = Quebra de linha
%.2f = Ponto flutuante = float - numero de casa decimais
%s = para colocar Texto
%d = para colocar números inteiros

ctrl + shift + F = Fazer auto indentação.

double x = 10.35784

System.out.println(x)
System.out.printf("%.2f%n", x) - Vai dar resultado de um número quebrado, até duas casas decimais.
System.out.printf("%.4f%n", x) - Vai dar resultado de um número quebrado, até quatro casas decimais.

println tem uma quebra de linha ao final.
print não tem quebra de linha no final.
printf quando querer colocar dados dentro do console.

import java.util.Locale; -> importar primeiro para depois colocar o Locale.
Locale.setDefault(Locale.US); -> Para setar o separador dos numeros com ponto ao invés de vírgula.

Casting é a conversao de valores. Exemplo código abaixo: 
		int l, m;
		double resultado;
		l = 5;
		m = 2;		
		resultado = (double) l / m;		
		System.out.println(resultado);

Coloca (double) na variável resultado, para que o programa faça a conversao de INT para Double.
